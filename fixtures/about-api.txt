## Who can use the Amazon Ads API?

The Amazon Ads API offers tailored solutions to streamline campaign management and maximize advertising impact. Typical users include:

* Advertising solution providers who integrate with the API and offer paid management and reporting tools to agencies and advertisers who lack the development resources and bandwidth to directly integrate with the Amazon Ads API
* Agencies that have internal engineering resources and that manage a significant volume of advertising campaigns for advertising clients
* Advertisers who manage their ad spend, have internal engineering resources, and directly run a significant volume of Amazon Ads campaigns

### Amazon Ads API FAQs

How much is the API fee?

There are no additional fees from Amazon Ads to use the API. The only fees include standard account fees to sell on Amazon, as well as the campaign costs for using Amazon Ads advertising products.

Will I be charged for ads that show on Amazon while testing the API?

We offer [test accounts](https://advertising.amazon.com/API/docs/en-us/guides/account-management/test-accounts/overview?ref_=a20m_us_p_api_faq_atc_tstacct) for development and testing purposes. Ads created in this environment will not show on Amazon, and therefore no fees will apply.

I'm an advertiser without much experience using APIs. How can I perform bulk changes to my Amazon Ads campaigns?

* If you are using a third-party solution to manage your selling on Amazon inventory, order, and pricing information, or are using an agency to manage your online advertising, we recommend asking your partner if they offer API integration in their solution. For a list of regions in which the Amazon Ads API is currently available to solution providers and advertising agencies, see the "[API Endpoints](https://advertising.amazon.com/API/docs/en-us/get-started/how-to-use-api?ref_=a20m_us_p_api_faq_atc_endpts)" section of "How to use the Amazon Ads API."
* Alternatively, you can use the [bulk operations](https://advertising.amazon.com/API/docs/en-us/bulksheets/no-code-overview?ref_=a20m_us_p_api_faq_atc_blkops) feature for sponsored ads and Amazon DSP. Sellers can access this in Seller Central, while vendors can access this in the advertising console.

Where can I find solution providers who offer access to API-integrated tools?

If you're looking for an agency to help plan and manage your Amazon Ads campaigns, or a tool provider that can offer unique technology solutions, you can turn to the [Amazon Ads partner directory](https://advertising.amazon.com/partners/directory?ref_=a20m_us_api_faq_prtdrct) for support. The partner directory is an online resource that makes it easy to find a partner that's right for your business. When searching on the partner directory, you can filter by criteria like "products," "marketplaces," or "service model" to identify a partner most relevant to your business.

Do you have a Developers Portal?

Yes, we have an [Amazon Ads API Developers Portal](https://advertising.amazon.com/API/docs/en-us/get-started/overview?ref_=a20m_us_api_faq_atc_dv). You'll find onboarding materials, documentation, release notes, and more. You can sign into the advanced tools center to see a personalized dashboard experience. We also offer a [Github](https://github.com/amzn/ads-advanced-tools-docs?ref_=a20m_us_p_api_faq_atc_gthb) repository for developers.

Who should apply for API access?

Developers interested in building applications using the Amazon Ads API. Advertisers looking to work with a third-party tool that uses the API do not need to request API access.
