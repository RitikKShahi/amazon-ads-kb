# Sponsored Display (Historical)

Sponsored Display was Amazon's self-service display advertising format that helped brands reach relevant audiences wherever they spent time using Amazon-generated creative or custom creative assets. It has been rebranded as **Display Ads** as part of Amazon's unified display advertising offering.

## Note

Sponsored Display has been **rebranded and is now called 'display ads'** as part of Amazon's unified display advertising offering. Existing campaigns continue to run uninterrupted, but new campaigns should be created through the unified platform's Display ad format.

# Details

## Historical Overview

Sponsored Display was Amazon's flexible display advertising solution that enabled brands to:
- Reach relevant audiences using Amazon's first-party insights
- Use Amazon-generated creative or custom creative assets
- Engage customers across Amazon properties and third-party destinations
- Support both image and video formats

## Platform Transition

### Rebranding

As part of Amazon's platform unification, Sponsored Display was rebranded:
- **New name**: Display Ads
- **Enhanced capabilities**: Integration with Amazon DSP workflows
- **Unified platform**: Single workspace for all Amazon Ads products

### Campaign Continuity

- **Existing campaigns**: Continue to run uninterrupted following the platform transition
- **Campaign access**: Users access existing Sponsored Display campaigns through Campaign Manager in the Ads Console
- **No disruption**: All campaign settings, bids, and targets are preserved

## Legacy Features

Sponsored Display provided advertisers with:

### Targeting Capabilities
- Audience-based behavioral targeting (not keyword-based)
- Contextual targeting for specific products or categories
- Remarketing based on views and purchases
- Amazon audience segments

### Pricing Models
- **Cost-Per-Click (CPC)**: Pay when customers click ads
- **Cost-Per-Thousand Viewable Impressions (vCPM)**: Pay for viewable impressions
- **No minimum spend requirements**
- **No management fees**

### Ad Formats
- Image ads with creative optimization
- Video ads for enhanced engagement
- Dynamic creative based on product detail pages
- Promotional badges (deals, coupons, Subscribe & Save)

### Performance Measurement
- Standard metrics: impressions, clicks, spend, CTR, CPC
- Performance metrics: sales, ACOS, ROAS
- Amazon-specific metrics: DPVR, ATL, new-to-brand

## Migration to Unified Platform

### Accessing Historical Campaigns

Historical Sponsored Display campaigns are accessed through:
1. Navigate to **Campaign Manager** in the Ads Console
2. View existing Sponsored Display campaigns
3. Manage bids, budgets, and targets as before

### Creating New Campaigns

New display advertising campaigns are created through the unified platform:
1. Navigate to **'Campaigns'** in the Ads Console
2. Click **'Create Campaign'**
3. Select **'Display'** campaign type
4. Configure targeting, budget, and creative

## Relationship to Display Ads

Sponsored Display has evolved into **Display Ads** with:
- **Same core capabilities**: Flexible targeting, multiple pricing models
- **Enhanced reach**: Integration with Amazon DSP supply
- **Unified interface**: Single platform for all display advertising
- **Preserved features**: All existing functionality maintained
