# Amazon Ads API Documentation

Amazon provides comprehensive documentation and developer resources to support API integration and ongoing development through the Amazon Ads API Developers Portal and associated platforms.

# Details

## Amazon Ads API Developers Portal

The Amazon Ads API Developers Portal serves as the central hub for developers and includes:

- **Onboarding materials** - getting started guides and setup instructions
- **API documentation** - detailed reference documentation for all endpoints and features
- **Release notes** - updates and changes to the API
- **Personalized dashboard** - customized experience based on developer's access and projects

## GitHub Resources

A GitHub repository is available for Amazon Ads API developers:

- **Repository**: `amzn/ads-advanced-tools-docs`
- **Purpose**: Additional documentation, code samples, and advanced tools
- **Access**: Publicly available for all developers working with the Amazon Ads API

## Partner Directory

The Amazon Ads partner directory helps advertisers find solution providers that work with the Amazon Ads API. Users can filter partners by:

- Products supported
- Marketplaces served
- Service model

## Developer Support

These resources provide developers with the information needed to:

- Understand API capabilities and limitations
- Implement API integrations correctly
- Stay updated with API changes and new features
- Access community support and best practices
