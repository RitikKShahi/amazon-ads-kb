# Amazon Ads API Testing

Test accounts are available for API development and testing, providing a safe environment for developers to build and test their API integrations without impacting live advertising campaigns.

# Details

## Test Account Features

Test accounts provide several benefits for API development:

- **Safe testing environment** - ads created in test accounts do not appear on Amazon's live platform
- **No fees incurred** - testing does not generate advertising costs
- **Full API functionality** - developers can test all API features and endpoints
- **Isolated from production** - test activities don't affect real advertising campaigns

## Use Cases

Test accounts are ideal for:

- Initial API integration development
- Testing new features or API endpoints
- Debugging applications without risk to live campaigns
- Training developers on API functionality
