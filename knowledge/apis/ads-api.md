# Amazon Ads API

The Amazon Ads API is a REST API that enables programmatic management of advertising operations for Amazon advertisers and Partner Network members. It provides direct access to Amazon's signals, supply, and technology for advertising management and reporting.

# Details

The Amazon Ads API enables management of advertising resources associated with an Amazon advertiser account. It enables planning, activating, and measuring programmatic media strategies on Amazon at scale. It provides programmatic access to multiple advertising products and services:

- Sponsored ads
- Streaming TV ads
- Amazon DSP (Demand-Side Platform)
- Amazon Marketing Cloud (AMC)
- Amazon Marketing Stream

## Key Capabilities

API integrations support building various advertising automation solutions:

- Custom reporting dashboards and detailed reporting
- Custom rules and automation logic
- Automatic bid optimizations
- Keyword optimizations
- Always-on budget optimization
- Automated campaign, budget, and bid optimizations
- Bulk operations for sponsored ads and Amazon DSP

The bulk operations feature is accessible via Seller Central for sellers and the advertising console for vendors.

## User Types

Three typical user types of the Amazon Ads API:

1. **Advertising solution providers** - third-party businesses building advertising automation tools
2. **Agencies with engineering resources** - managing advertising for multiple clients
3. **Advertisers with engineering resources** - managing their own advertising accounts

## Cost

There are no additional fees from Amazon Ads to use the API beyond standard Amazon selling account fees and advertising campaign costs.

## Regional Availability

The Amazon Ads API is available to solution providers and advertising agencies in multiple regions, with specific API endpoints documented for each region in the official documentation.
